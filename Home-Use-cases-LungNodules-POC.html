Lung nodule use case

This model represents a proof of concept use case consisting in an AI detection of lung nodules in CT scans, which provides quantitative and qualitative assessments about the lesions. The modeling involves capturing the lesions as a body structure instance, allowing follow up over time.

AIRBodyStructure

AIRBodyStructure profile captures the anatomical location and the type of lesion from which the meassurements are taken.

  • .location is bound to AIRLungAnatomyValueSet, that contains lobes and segments of both lungs.
  • .morphology has binding to AIRLesionType with possible lesions detected in the lungs. In this use case the value is "http://snomed.info/sct"|"4147007"|"Mass (morphologic abnormality)"

AIRObservationRoot

This profile represents the finding itself, and holds the reference to the observation resources that contain the quantitative and qualitative meassurements of that finding. In this use case, AIRObservationRoot represents the instance of the Lung Mass (nodule) that is being assessed by the AI in this particular imaging study.

  • .code has a fixed value of SCT | 404684003 | Clinical finding.
  • .valueCodeableConcept is populated with the type of lesion detected, matching BodyStructure.morphology. In the example below the code is "http://snomed.info/sct"|"4147007"|"Mass (morphologic abnormality)".
  • .hasMember references observation resources that contain quantitative and qualitative values.
